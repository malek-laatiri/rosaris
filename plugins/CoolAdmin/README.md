DESCRIPTION
-----------

Based on the FlatSIS theme (core).

Inspired by the CoolAdmin template by Colorlib (MIT license):

- https://github.com/puikinsh/CoolAdmin
- https://colorlib.com/polygon/cooladmin/index.html

See the `Gruntfile.js` file for more information about how the `stylesheet.css` file is minified.

Includes the _Poppins_ font (SIL Open Font License).


INSTALL
-------
Copy the `CoolAdmin/` folder (if named `CoolAdmin-main`, rename it) and its content inside the `assets/themes/` folder of RosarioSIS.

Go to _School > Configuration_ and select it as **Default Theme**.

Please remember users can choose their preferred theme by going to _Users > My Preferences_.

Requires RosarioSIS 10.0+
